# Privacy Policy

**Cluster Auto** is built with user privacy as the absolute highest priority. 

## 🔒 Personal Data Collection
* **Zero Tracking**: We do not collect, monitor, store, or transmit any personal data.
* **No Telemetry**: There is no telemetry code, analytics suite, or usage tracking built into the app.
* **No Network Logs**: The application does not connect to external servers or send diagnostic reports.

## 📂 Local Permissions
Cluster Auto operates as a local background daemon service. All root/shell-level access is performed strictly on-device at the request of authorized applications. No data is ever uploaded.
